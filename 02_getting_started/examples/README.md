   - Anlegen von Diagrammen

1. [Aufruf einer HTTP REST API](./rest_api/README.md)

   - Zugriff auf Token

1. [Erstellung eines Chatbots](./chatbot/README.md)

1. [Turmstapel-Problem](./towers/README.md)
